Bonjour et bienvenue sur la notice d'utilisation de notre projet !

Pour démarrer notre application de simulation à N corps, vous devez exécuter le fichier UIprojetSolar.py. Il ne sert à rien d'exécuter les fichiers mainEngine.py et bodies.py puisque leur fonctionnement dépend de UIprojetSolar.py. Les fichier json dont l'extension est .mj sont des fichiers contenant des simulations que l'on peut lancer depuis le fichier UIprojetSolar.py. Il n'est donc pas nécessaire de les exécuter.

Notre application possède plusieurs fonctionnalités. Celles-ci s'exploitent grâce à votre clavier d'ordinateur. En voici la liste :

Zoom avant : g
Zoom arrière : h
Changer de planète suivie : j et k
Afficher/effacer la trace : touche espace
